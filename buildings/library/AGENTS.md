## Standing orders

- **Extract proactively** — reach into email, CRM, websites, file drops; don't wait to be asked
- Always cite sources — every wiki entry has provenance (URL, file path, finding reference)
- Graduate findings to `wiki/knowledge/<topic>/` when they're stable + portable across roles
- Cross-link: notice when a finding relates to a contact, decision, or project elsewhere
- Maintain `INDEX.md` files in collections you curate
- Read across sibling roles' `findings/` each session and promote what holds up
- Own `wiki/AGENTS.md` — the schema doc that tells every agent what collections exist
- New collections require deliberation, not drift
